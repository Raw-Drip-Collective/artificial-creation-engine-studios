# Contributing to Artificial Creation Engine Studios

Thank you for your interest in contributing to Artificial Creation Engine Studios!

## How to Contribute

### Reporting Bugs

1. Search existing issues to avoid duplicates
2. Create a new issue with:
   - Clear title describing the bug
   - Steps to reproduce
   - Expected vs actual behavior
   - Environment details

### Feature Requests

1. Check existing feature requests
2. Open a new issue with:
   - Clear description of the feature
   - Use cases
   - Potential implementation approaches

### Pull Requests

1. Fork the repository
2. Create a feature branch:
   ```bash
   git checkout -b feature/your-feature-name
   ```
3. Make your changes following the code style
4. Add tests if applicable
5. Commit and push:
   ```bash
   git add .
   git commit -m 'Add: Your feature description'
   git push origin your-branch-name
   ```
6. Open a Pull Request

## Development Setup

```bash
# Clone repository
git clone https://github.com/Raw-Drip-Collective/artificial-creation-engine-studios.git
cd artificial-creation-engine-studios

# Install dependencies
npm install
# or
pnpm install

# Start development server
npm run dev
```

## Code Style

- Follow ESLint and Prettier for code formatting
- Use TypeScript for type safety
- Write meaningful variable and function names
- Add comments for complex logic

## Testing

```bash
# Run tests
npm test

# Run type checking
npm run type-check
```

## Questions?

Open an issue for questions about contributing.

Thank you for contributing!
